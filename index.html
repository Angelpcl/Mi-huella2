Política de Privacidad — Mi Huella

Última actualización: 21 de octubre de 2025

Mi Huella (en adelante, la Aplicación, «nosotros» o «nuestro») se compromete a proteger la privacidad de las personas que usan la aplicación. Esta Política de Privacidad explica qué datos recopilamos, para qué los usamos y cómo los protegemos.

1. Información que recopilamos

La aplicación puede recopilar los siguientes tipos de información:

  • Ubicación geográfica: requerida para mostrar la posición actual del usuario en el mapa, localizar veterinarias y permitir funciones de rastreo de mascotas mediante servicios de terceros (por ejemplo, Traccar).
  • Datos técnicos: información no personal como versión del sistema operativo, modelo del dispositivo y registros de errores, que se usan para mejorar la estabilidad y rendimiento de la aplicación.

Por defecto, la aplicación no solicita ni almacena datos personales identificables (p. ej., nombre, dirección, correo electrónico o teléfono) salvo que el usuario los proporcione voluntariamente en funciones futuras.

2. Finalidad del tratamiento

Utilizamos la información recopilada con los siguientes fines:

  • Mostrar la ubicación actual del usuario en el mapa.
  • Localizar y mostrar veterinarias y establecimientos cercanos.
  • Facilitar el rastreo de mascotas a través de integraciones con servicios externos (por ejemplo, Traccar).
  • Mejorar la experiencia del usuario y la estabilidad de la aplicación mediante análisis de errores y desempeño.

3. Compartición y terceros

No compartimos los datos personales de los usuarios con terceros para fines propios. Sin embargo, la aplicación interopera con servicios externos para ofrecer funcionalidades:

  • Google Maps API: usada para mostrar mapas y realizar búsquedas de lugares.
  • Traccar API: usada cuando el usuario habilita funciones de rastreo de dispositivos o mascotas.

Estos proveedores tienen sus propias políticas de privacidad y prácticas de tratamiento de datos; recomendamos revisar sus términos antes de usar las funciones asociadas:

  • Política de Privacidad de Google
  • Política de Privacidad de Traccar

4. Permisos del dispositivo

Para ofrecer todas las funciones, la aplicación puede solicitar permisos en el dispositivo:

  • Acceso a la ubicación: imprescindible para mostrar la posición en el mapa y para el rastreo de mascotas. El usuario puede permitir el acceso siempre, solo cuando la app esté en uso, o denegarlo.
  • Acceso a Internet: necesario para comunicarse con APIs externas y mostrar mapas.

El usuario puede revocar estos permisos desde la configuración del sistema en cualquier momento; si lo hace, algunas funciones pueden dejar de funcionar correctamente.

5. Seguridad

Implementamos medidas técnicas y organizativas razonables para proteger la información frente al acceso no autorizado, pérdida o divulgación. No obstante, ningún sistema es completamente invulnerable; la transmisión de datos por Internet conlleva riesgos inherentes.

6. Conservación de los datos

Los datos de ubicación y técnicos utilizados para mostrar información en tiempo real no se almacenan de forma persistente en los servidores de la Aplicación salvo que el usuario active explícitamente funciones de rastreo que requieran almacenamiento en un servicio tercero (p. ej., Traccar). Si en el futuro se decide almacenar datos de forma persistente, se informará al usuario y se solicitará su consentimiento cuando sea necesario.

7. Derechos del usuario

El usuario tiene los siguientes derechos respecto a sus datos:

  • Revocar permisos de ubicación desde la configuración del dispositivo.
  • Desinstalar la aplicación para dejar de compartir datos con la misma.
  • Solicitar información sobre qué datos se han compartido con servicios terceros (si aplicara).

8. Cambios en esta política

Podemos actualizar esta Política de Privacidad de vez en cuando. Cuando haya cambios relevantes, se indicará la nueva fecha de última actualización en la parte superior de este documento.
